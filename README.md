## Development Setup

1. Clone the repository:
```bash
git clone <repository-url>
cd gym-management
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the development server:
```bash
python app.py
```

4. Access the application at `http://localhost:5000`
